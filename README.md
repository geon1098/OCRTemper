# 온도관리 게시판

<h2>프로젝트 설명</h2>
병원에서 혈액관리에서 수기로 하는 병원들이 많다고 하여
편하게 갱신을 할 수 있게 하는 '온도 관리 게시판' 입니다.
카메라로 캡쳐하면 09시, 13시, 17시 간격으로 가장 가까운 시간대에
온도를 갱신할 수 있는 관리 게시판 입니다.
